Image Mosaicing

Abstract

This project focusses on the implementation of various conceptual aspects related to Image Stitching using various techniques, such as planar Homography as well as Cylindrical Image Stitching using the translational model. Given an input video sequence, we take the images in pairs, find the features using various novel techniques such as SIFT, SURF, ORB, etc., then perform feature matching techniques using the Brute Force KNN method. In one case, we estimate the Homography Matrix between the two images used for the geometric transformation of one of the images to stitch the two images together following the matched features. In another case, we first convert the images obtained in the cylindrical domain and perform feature detection, followed by feature matching. We then perform image stitching using the Translational model, where the only geometric translation of the image is done to perform the stitching.

Number Plate Detection

Abstract

The project focusses on the detection and registration of vehicles and their number plates. The camera feed is first processed to detect the vehicle type present using YOLOv4 architecture, trained on COCO dataset and finetuned on some of our dataset. Then the number plate present is detected and segmented using a different YOLOv4 architecture, trained on a mix of OID dataset as well as some of our own. Finally, from the segmented number plate, individual characters are segmented using multiclass classification with YOLOv4, trained entirely on our dataset. The solution was then integrated with web and android application. This project was done as a problem statement provided by MixOrg Consulting Services Pvt. Ltd. under MHRD and MIC organized Smart India Hackathon, 2020 (Software Edition), where our solution secured first position and a cash prize of Rs. 1,00,000.

Stereo 3D Reconstruction

Abstract

The objective is to reconstruct a 3D equivalent, given the pair of images. Features are extracted for the two images using Scale Invariant Feature transform (SIFT), followed by feature matching using Brute Force KNN and outlier rejection. The essential matrix is calculated for the point correspondence and outliers are rejected again using RANSAC. Using SVD, pose is recovered. From the recovered pose, stereo rectification is performed such that the epipoles for both the frames tend to infinity. Using Semi Global Block Matching (SGBM), the disparity map is calculated, for left image with respect to right image and vice versa. Using these two maps, Weighted Least Square (WLS) Filtering is performed to reduce speckles and for uniform and smooth continuity of map. A reprojection matrix Q is defined to project these point onto 3D, in order to make the final point cloud. Currently, this concept is being extended to Structure from Motion (SFM).

Visual Odometry

Abstract

The project aims at understanding the working of visual odometry. An odometer is a small gadget that tells you how much distance it has traveled on the robot or car to which it is attached to. How it works internally varies different types of gadgets and on the robot to which it is attached to. In robotics, a more generalized definition of odometry is considered, estimating the trajectory of the moving robot, along with the distance traveled. So, it is easy to see that for each position of the robot or car, there is an associated vector showing its pose in terms of cartesian coordinates and Euler angles. In this project, we aim at understanding at doing the same using a camera. So, given the input trajectory of the robot, we are required to construct its 6 DOF trajectory. If we use a single camera, it is known as Monocular Visual Odometry and if we are using two or more, then it is termed as Stereo Visual Odometry. The project is currently in progress.

American Sign Language Detection

Abstract

This was a project done for selection to Technoseason 1 Image Processing Task for IvLabs. We had to make a program that can take the input hand gestures as our input and give its equivalent meaning in terms of American Sign Language Digits. There are various approaches to tackle this problem, however, I went through a more traditional route. The image was taken, converted to HSV to perform the color thresholding of the skin. The contour was obtained and the convex hull was performed to find the defects in the contour. Those defects were filtered using different equations and constraints. To make the corners sharper, the hand was eroded to find the distinct points. The code was written in python3. This solution made our team stand second in the Technoseason Recruitment Drive.

Cursor Control Using Face Gestures

Abstract

This project presents the software implementation for the detection of facial landmarks with the data trained on iBUG 300 – W dataset by dlib library in python and implementing them for controlling mouse operations. The proposed method aims at catering to the needs of differently – abled people (for example, people with locked – in syndrome), who are not able to operate a computer. Blinking of the left eye will result in a left click, the blinking of the right eye will result inright-click and scroll mode is enabled by opening the mouth. The same is done for disabling the scroll mode. Concept of Eye Aspect Ratio in the case of eyes and mouth aspect ratio in the case of mouth has been used for checking whether that particular mouse operation is performed or not. Image enhancement has been done primarily by Contrast Limited Adaptive Histogram Equalization (CLAHE) and Gaussian Filtering.
